# Texture Drop-In Guide

Place your texture images in this folder with these exact file names:

- ceiling.png
- floor.png
- wall-far.png
- wall-mid.png
- wall-near.png
- wall-side.png
- stairs.png

Notes:
- PNG works best for transparent pixels.
- Keep textures tileable when possible for cleaner results.
- If any file is missing, the game falls back to built-in solid colors.
- You can change file names/paths in js/main.js under TEXTURE_PATHS.
